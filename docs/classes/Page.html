Page extends ArrayObject
in package

Page.php : 10

A container that holds data needed to create a new page.

This class acts like an array, whose keys define the page properties. Then we can either create an HREF so that it can be accessed by a future http request (via the Smr\Session), or forwarded to within the same request.
