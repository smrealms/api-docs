public close() : mixed

Closes the connection to the MySQL database. After closing this connection, this MySqlDatabase instance is no longer valid, and will subsequently throw exceptions when attempting to perform database operations.

You must call MySqlDatabase::getInstance() again to retrieve a valid instance that is reconnected to the database.
